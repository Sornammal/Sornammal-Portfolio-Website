About Me

Hi there, my name is Sornammal Madhavan.

I’m a graduate of Government College for Women’s (A), Kumbakonam, with a bachelor’s degree in Mathematics. My academic background sharpened my analytical thinking and problem-solving abilities, which laid the foundation for my passion for software development. Over time, I transitioned into full stack development, gaining strong hands-on experience in building responsive and interactive web applications using technologies such as HTML, CSS, JavaScript, React.js, Node.js, Express.js, and MongoDB.

Professionally, I’ve worked across automation and data privacy domains. At Avasoft, I served as an Automation Engineer, contributing to RPA initiatives using UiPath to automate business processes and streamline testing. Currently, at Solvation LLC, I work as a Privacy Specialist, ensuring data compliance and managing risks in alignment with regulations such as GDPR and CCPA. These roles have strengthened my ability to collaborate, adapt, and solve problems in dynamic environments.

I’ve also developed and deployed multiple real-time projects that reflect my technical skills and creativity. These include a pixel-perfect Apple Homepage Clone built with React and Vite, a responsive e-commerce website called Nostra using Tailwind CSS, and a Weather API Application that fetches real-time data using OpenWeatherMap API. This weather app allows users to search for any city and view live data including temperature, humidity, and general conditions—all presented in a clean and user-friendly interface.
